# @umijs/plugin-dynamic-api-routes

Umi plugin for dynamic creation of routes called from api.

## Install

```bash
pnpm i @umijs/plugin-dynamic-api-routes
```

## Usage

Configure in `.umirc.ts`,

```js
export default {
  plugins: [
    ['@umijs/plugin-dynamic-api-routes'],
  ],
}
```
